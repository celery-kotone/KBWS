
                                 kclustalw



Function

   Multiple sequence alignment using ClustalW 2

Description

   kclustalw calculates global multiple sequence alignment of DNA and
   protein sequence utilizing ClustalW 2.
   ClustalW 2 is a general purpose multiple sequence alignment program
   for DNA or protein. It calculates the best match for the selected
   sequence, and lines them up so that identifers, similarities, and
   differences can be seen.

   kclustalw utilizes ClustalW 2 SOAP service provided by EBI (European
   Bioinformatics Institute). Original web-service is located at the
   following URL:

   http://www.ebi.ac.uk/Tools/clustalw2/index.html

   This tools is a subset of Keio Bioinformatics Web Service (KBWS) package,
   which contains interfaces to bioinformatics web services through a proxy
   server at Keio University. kclustalw is thus an interface of "runClustalw"
   method included in KBWS SOAP service.
   This method can be alternatively accessed directly from programming languages
   as SOAP web service. Please refer to the KBWS online documentations
   http://soap.g-language.org/kbws/ for more information.

Command line arguments


   Standard (Mandatory) qualifiers:
  [-sequence]          seqall     (Gapped) sequence(s) filename and optional
                                  format, or reference (input USA)
  [-outfile]           outfile    [.aln] multiple aligned sequence file
  [-dndoutfile]        outfile    [.dnd] Dendrogram (tree file) from clustalw output
                                  file

   Additional (Optional) qualifiers: (none)
   Advanced (Unprompted) qualifiers:
   -alignment          string     [full] method used for pairwise alignment stage
   -outformat          string     [aln1] Alignment output format. Values: 'aln1',
                                  'aln2', 'gcg', 'phylip', 'pir' or 'gde'
   -matrix             string     protein scoring matrix
   -outorder           string     order of sequences in the alignment
   -ktup               integer    word size
   -window             integer    window size around best diagonals
   -gapopen            integer    gap creation penalty
   -gapext             float      gap extension penalty
   -gapdist            integer    gap separation penalty
   -endgaps            boolean    [N] Do not score end gaps
   -pairgap            integer    gap penalty
   -topdiags           integer    top diagonals to consider
   -score              string     score type
   -tossgaps           boolean    [N] discard gap regions when calculating tree
   -kimura             boolean    Use Kimura's correction
   -outputtree         string     ['none'] Tree format. Values: 'none', 'nj', 'phylip' or 'dist'
   -tree               boolean    [N] generate phylogenetic tree
   -quicktree          boolean    [N] perform 'fast' alignment
   -showalign          boolean    [N] perform alignment
   -clustering         string     Tree type. Values: 'nj' or 'upgmp'
   -numiter            integer    number or iterations
   -iteration          string     ['none'] iteration type. Value: 'none', 'alignment' or 'tree'
